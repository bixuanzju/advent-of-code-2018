data API :: Bool -> Type -> Type where Source #

An API command. An API k a an AoC API request that returns results of type a; if k is 'True, it requires a session key. If k is 'False, it does not.

Session Keys

data SessionKey :: Bool -> Type where Source #

Holds a session key. SessionKey 'True contains a session key for sure, but SessionKey 'False may or may not contain one.

Constructors

HasKey :: String -> SessionKey k 
NoKey :: SessionKey False 

sessionKey :: API k a -> Maybe String -> Maybe (SessionKey k) Source #

Wrap a Maybe String (a possible session key) into a SessionKey based on the expectations of a given API command. If the API command requires a key, Nothing will be returned if no key is given. If the API command doesn't require a key, Just will always be returned.

sessionKey_ :: Maybe String -> SessionKey False Source #

Convert a Maybe String (a possible session key) into a SessionKey 'False --- a SessionKey that may or may not contain a key.
